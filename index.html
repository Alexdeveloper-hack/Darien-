Conceptos y Descripciones

Concepto Descripción Concepto Descripción
Tabla en HTML Estructura que organiza datos en filas y columnas. Formulario en HTML Conjunto de elementos que permiten la entrada de datos del usuario.
Etiqueta table Define el inicio y el final de una tabla. Etiqueta form Elemento que contiene controles de entrada para enviar datos.
Atributos de table Propiedades como border, cellpadding y cellspacing para el estilo. Etiqueta input Campo que permite al usuario introducir datos.
Etiqueta tr Define una fila dentro de la tabla. Etiqueta label Asocia un texto descriptivo a un campo de formulario.
Atributo tr Atributos como bgcolor y align para personalizar el estilo de la fila. Etiqueta select Permite al usuario seleccionar un valor de una lista desplegable.
Etiqueta td y th td define una celda de datos, y th define una celda de encabezado. Etiqueta textarea Campo que permite introducir texto de múltiples líneas.
Atributos de td y th Incluir colspan, rowspan y align para modificar la presentación. Etiqueta fieldset Grupo de elementos de formulario para organización visual.
Tablas avanzadas Uso de rowspan y colspan para diseños más complejos. Etiqueta legend Proporciona un título para un fieldset.
Etiqueta thead Sección que agrupa los encabezados de la tabla. Etiqueta meter Representa una medida dentro de un rango conocido.
Etiqueta tfoot Sección que agrupa el pie de tabla, usado para resúmenes. Etiqueta progress Indica el progreso de una tarea en curso.
Etiqueta tbody Define el cuerpo de la tabla con las filas de datos. Etiqueta datalist Proporciona una lista de opciones para un campo de entrada.
Etiqueta summary Proporciona un resumen de un contenido colapsable.
